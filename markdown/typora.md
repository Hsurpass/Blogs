# 修改typora快捷键

打开偏好设置 -> 通用 -> 打开高级设置：

打开`conf.user.json` 文件，修改`keyBinding`字段，例：

```json
// Custom key binding, which will override the default ones.
"keyBinding": {
  // for example: 
  // "Always on Top": "Ctrl+Shift+P"
  // "保持窗口在最前端": "Ctrl+Shift+P"
  "高亮": "Ctrl+H"
},
```
